# ユビキタス言語
- ブロガー (blogger)
  - ブログの記事を書いたり読んだりする人

- ビジター (visitor)
  - ブログの記事を読む人

- 記事 (article)
  - ブログの記事

- コメント (comment)
  - 記事に対するコメント

- ビジターコメント設定 (visitor-comment-setting)
  - ビジターが記事に対してコメントできるかどうかの設定

- ブロガーコメント設定 ( blogger-comment-setting )
  - ブロガーが記事に対してコメントできるかどうかの設定

- タグ (tag)
  - 記事の内容を表すキーワード

- ナイス (nice)
  - 記事の内容が役に立ったり、共感したことを表す
